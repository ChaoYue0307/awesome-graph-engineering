# Support

Thanks for using Awesome Graph Engineering. Here is where to go for help.

## Questions and discussion

- Open a [GitHub Discussion](https://github.com/ChaoYue0307/awesome-graph-engineering/discussions) for questions about the field, resource suggestions you want feedback on, or anything that is not a concrete change request.

## Corrections and additions

- A link is broken, a description is wrong, or a project's status changed → open an issue with the **Correction** template.
- You want to add a resource → read [CONTRIBUTING.md](CONTRIBUTING.md) first, then open an issue with the **Add a resource** template or send a pull request directly.

## Security

- See [SECURITY.md](SECURITY.md). Do not open public issues for security-sensitive reports.

## Response expectations

This is a community-maintained list. Issues and pull requests are usually reviewed within a week. Well-formed pull requests that follow the entry format merge fastest.
